# Mouse Jiggler 中文版

这是 Mouse Jiggler 的非官方简体中文本地化版本，基于原项目 [arkane-systems/mousejiggler](https://github.com/arkane-systems/mousejiggler)。

感谢原作者维护这个轻量、实用的 Windows 鼠标防空闲工具。本分支主要面向中文用户，保留原有功能，只对界面文本、托盘提示、模式名称和程序元数据做中文化处理。

## 功能简介

Mouse Jiggler 可以定时模拟鼠标输入，避免系统因为空闲而触发屏幕保护、锁屏或某些长时间任务中的空闲检测。

常见用途包括：

- 长时间安装、下载、监控任务时避免系统进入空闲状态
- 临时阻止屏幕保护或自动锁屏
- 在不方便修改系统电源设置时做轻量替代

请注意：它不是用来规避公司 IT 管理或监控的工具。原项目 README 也明确提醒，这类软件很容易被监控软件识别。

## 中文化内容

## 使用方法

下载或构建后运行 `MouseJiggler.exe`。

勾选“启用晃动”即可开始模拟鼠标输入，取消勾选即可停止。

常见选项：

- `设置...`：展开或收起详细设置
- `间隔(秒)`：设置每次晃动的间隔
- `距离倍率`：设置鼠标移动距离倍率
- `随机间隔`：在设定间隔内随机触发
- `启动时最小化`：启动后直接缩到托盘

晃动模式：

- `普通`：鼠标指针轻微移动
- `禅模式`：尽量只模拟输入，不明显移动指针
- `圆形`：按圆形轨迹移动
- `线性`：水平往返移动

## 构建

需要安装 .NET SDK。然后在仓库根目录运行：

```powershell
dotnet build MouseJiggler/MouseJiggler.csproj -c Release
```

构建产物会输出到 `MouseJiggler/bin/Release/` 下对应的目标框架目录。
